# Summary

BeamFi Protocol has 3 main components:

- [Beam Out (Link) API](/docs/smartcontracts/beamout/unique)
- [Beam API](/docs/smartcontracts/beam/create)
- [Beam Escrow API](/docs/smartcontracts/beamescrow/simplebeam)

Please follow each of the API links above to learn more about how to interact with the BeamFi Protocol.

Most parts of the API follow the [Blockchain pattern](/docs/smartcontracts/patterns)
