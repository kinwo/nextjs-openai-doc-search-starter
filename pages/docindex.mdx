# Home

Welcome to BeamFi Protocol Developer Documentation!

BeamFi is a decentralized DeFi protocol for micro payments running on [Dfinity](https://dfinity.org)'s [Internet Computer](https://internetcomputer.org) blockchain.
It allows users to send or receive a constant stream of micro payments.

It makes **real-time money streaming** possible by using the native features of Internet Computer blockchain:

- **Transfer of Value**
- **Self-custody** of assets through **Autonomous** BeamFi smart contracts
- [**Multi-block Transactions**](https://internetcomputer.org/capabilities/multi-block-transactions)

## History of BeamFi

BeamFi has been in development since 2022. It was originally called Beam and was built as part of Content Fly Escrow Payment.
To learn more about the history, ideas and vision of BeamFi, please read [What is BeamFi?](/whatis/features).
